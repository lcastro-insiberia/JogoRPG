# JogoRPG

Criar Jogo de RPG, com personagens e seus respectivos atributos 
utilizando os pilares da programação Orientada a Objeto no Csharp.

Sendo os quatro pilares da OOP:
H.E.P.A
Herença
Encapsulamento
Polimorfismo
Abstração
